智能台灯学习指南

本文档为智能台灯项目的完整学习路径指南，帮助复刻者从零基础开始，逐步掌握所有必要的技术栈，最终完成智能台灯项目的开发。

1. 项目简介​

1.1 项目目标​

本项目实现了一款具备环境感知与智能决策能力的台灯系统，具备以下核心功能：

  • 🔍 人体检测与跟随：通过 YOLOv8 实时检测人体姿态
  • 😊 人脸跟随：使用 Hopenet 模型实现人脸角度估计
  • 💡 智能照明：根据用户行为自动调节灯光
  • ☁️ 云端控制：通过涂鸦云平台实现远程控制
  • 🤖 机械臂控制：基于 LeRobot 框架的机械臂运动控制

9. 常见问题​

9.1 环境配置问题​

Q：串口连接不上怎么办？

  • 检查 USB 转串口驱动
  • 确认波特率设置（通常 115200）
  • 尝试更换 USB 接口

Q：SSH 连接失败？

  • 确认开发板 IP 地址
  • 检查网络连接
  • 验证 SSH 服务状态

9.2 驱动开发问题​

Q：驱动编译失败？

  • 确认内核版本匹配
  • 检查编译环境
  • 查看错误日志

Q：WS2812 不亮？

  • 检查 GPIO 引脚配置
  • 确认 DMA 通道可用
  • 示波器测量信号输出

9.3 AI 部署问题​

Q：模型转换失败？

  • 检查 ONNX 模型有效性
  • 确认输入输出节点名称
  • 查看转换日志错误信息

Q：推理速度慢？

  • 启用 INT8 量化
  • 优化后处理代码
  • 检查 NPU 占用情况

9.4 云端集成问题​

Q：Socket 连接失败？

  • 确认服务器 IP 和端口
  • 检查防火墙设置
  • 验证网络连通性

Q：APP 控制延迟高？

  • 优化 Socket 心跳间隔
  • 减少数据发送频率
  • 检查网络质量
