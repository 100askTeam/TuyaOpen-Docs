YOLOV13模型部署

参考资料：

  • iMoonLab/yolov13: Implementation of "YOLOv13: Real-Time Object Detection with Hypergraph-Enhanced Adaptive Visual Perception".
  • kaylorchen/ai_framework_demo: 针对于ai_framefork的测试demo

源码获取链接: https://pan.baidu.com/s/1ZqppMvmww0FPIrp7dJYBgA?pwd=45r3 提取码: 45r3

  • yolov13.zip：yolov3源码压缩包，包含模型、ONNX导出程序等。
  • rknn-yolov13.tar.gz：RKNN模型转换程序、端侧推理程序。

1.获取模型​

通过Git下载模型源码：

git clone https://github.com/iMoonLab/yolov13.git

在Linxu x86平台使用Conda创建虚拟环境：

conda create -n yolov13 python=3.11
conda activate yolov13

手动下载flash-attn等库。

wget https://github.com/Dao-AILab/flash-attention/releases/download/v2.7.3/flash_attn-2.7.3+cu11torch2.2cxx11abiFALSE-cp311-cp311-linux_x86_64.whl

安装YOLOV3依赖库:

pip install -r requirements.txt

可编译方式安装库

pip install -e .

下载预训练模型：YOLOv13-N YOLOv13-S YOLOv13-L YOLOv13-X，这里使用YOLOV13-N。

2.修改源码​

修改导出ONNX模型的检测头，源码位置：ultralytics/nn/modules/head.py，将原来的

 def forward(self, x):
        """Concatenates and returns predicted bounding boxes and class probabilities."""
        if self.end2end:
            return self.forward_end2end(x)

        for i in range(self.nl):
            x[i] = torch.cat((self.cv2[i](x[i]), self.cv3[i](x[i])), 1)
        if self.training:  # Training path
            return x
        y = self._inference(x)
        return y if self.export else (y, x)

修改为：

 def forward(self, x):
        # 导出 onnx 增加
        y = []
        for i in range(self.nl):
            t1 = self.cv2[i](x[i])
            t2 = self.cv3[i](x[i])
            y.append(t1)
            y.append(t2)
        return y

3.新增导出程序​

在项目的源码目录下，新建export_onnx.py，填入以下内容：

from ultralytics import YOLO
model = YOLO('yolov13n.pt')  # Replace with the desired model scale
model.export(format="onnx", half=True)

执行程序：

python3 export_onnx.py

执行完成可以在当前目录下看到导出的yolov13n.onnx模型：

4.简化ONNX模型​

安装ONNX库：

pip install onnxsim

执行模型简化：

python -m onnxsim yolov13n.onnx yolov13n-sim.onnx --input-shape 1,3,640,640

5.模型转换​

将ONNX模型、测试图像、测试图像路径文件、模型转换程序：

baiwen@dshanpi-a1:~/rknn-yolov13$ ls
bus.jpg  convert.py  dataset.txt  yolov13n-sim.onnx

激活模型转换环境：

conda activate rknn-toolkit2

执行模型转换程序：

python3 convert.py ./yolov13n-sim.onnx rk3576
